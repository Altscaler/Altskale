# Magic KB Installation Guide

## Loading the Extension in Chrome

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" in the top right corner
3. Click "Load unpacked"
4. Select the `/home/user/Altskale` directory (or wherever you cloned this repo)
5. The Magic KB extension should now appear in your extensions list

## Using Magic KB

### Quick Start
1. Click the Magic KB icon in your Chrome toolbar
2. Browse the **Prompt Library** to see 50+ pre-built prompts across categories:
   - Social Media
   - Sales & Outreach
   - Design
   - Product Management
   - Marketing & SEO
   - Writing & Content
   - Education & Learning
   - Customer Support
   - Data & Analysis
   - AI & Prompting
   - Meetings & Collaboration
   - Personal Productivity

3. Click any prompt to view and edit it
4. Click "Save" to add it to your personal shortcuts
5. Type the shortcut (e.g., `/lipost`) followed by **space** in any text field to expand it

### Creating Custom Shortcuts
1. Click the **+** button in the popup
2. Enter your shortcut (e.g., `/myshortcut`)
3. Add a note (optional)
4. Enter the text you want it to expand to
5. Click "Save"

### Managing Shortcuts
- **Search**: Use the search box to find shortcuts quickly
- **Edit**: Click any shortcut to edit it
- **Delete**: Click "Delete" when editing a shortcut
- **Export/Import**: Use the "Export" and "Import to library" buttons in the Options page

### Filtering the Prompt Library
- Click any category tag to filter prompts by category
- Use subfilters:
  - **In KB**: Only show prompts already in your keyboard shortcuts
  - **Not in KB**: Only show prompts not yet added
  - **Short**: Prompts under 200 characters
  - **Long**: Prompts over 200 characters

## Tips

- Shortcuts trigger on **space** or **semicolon** (`;`)
- Use `/` prefix for easy organization (e.g., `/cold`, `/prd`)
- Add notes to remember what each shortcut does
- Export your shortcuts regularly as a backup
- Browse the Prompt Library for inspiration
