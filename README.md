# Magic KB

A Chrome extension that turns keyboard shortcuts into full text expansions. Built-in library of 50+ professional prompts for AI, content creation, sales, product management, and more.

## Quick Start

1. Load the extension in Chrome (see [INSTALL.md](INSTALL.md))
2. Click the Magic KB icon in your toolbar
3. Browse the **Prompt Library**
4. Click any prompt to add it to your shortcuts
5. Type the shortcut + space in any text field
